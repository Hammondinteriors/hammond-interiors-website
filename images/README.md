# Images Directory

Place your Hammond Interiors project images here.

## Folder Structure

```
images/
├── hero/
│   └── hero-bg.jpg          - Hero section background (1920x1000px)
├── gallery/
│   ├── kitchen-1.jpg        - Modern kitchen project
│   ├── kitchen-2.jpg        - Contemporary kitchen project
│   ├── bathroom-1.jpg       - Luxury bathroom project
│   ├── bathroom-2.jpg       - Ensuite bathroom project
│   ├── renovation-1.jpg     - Full property renovation
│   └── renovation-2.jpg     - Kitchen & bathroom suite
└── about/
    └── team.jpg             - Team or installation photo (400x500px)
```

## Image Specifications

- **Format**: JPG (JPEG) recommended for photos
- **Optimization**: Compress to 80-85% quality for web
- **File sizes**: Keep under 150KB for gallery images
- **Dimensions**:
  - Hero: 1920x1000px
  - Gallery: 800x600px
  - About: 400x500px

## How to Add Images

1. Download images from Facebook/Instagram
2. Optimize using TinyPNG.com or similar
3. Save to appropriate folder with correct filename
4. Images will automatically display on the website

See `ADD_IMAGES.md` in the root directory for detailed instructions on downloading and optimizing images.
